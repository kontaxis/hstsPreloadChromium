# hstsPreloadChromium

```
usage: hstsPreloadChromium.py [-h] [--verbose] E [E ...]

Look up entries in the Chromium HSTS preload list.

positional arguments:
  E              Entry to look up.

optional arguments:
  -h, --help     show this help message and exit
  --verbose, -v  Output information on the lookup process.
```
